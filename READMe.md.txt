#  Interactive Study Planner

A visually interactive productivity web app that helps students manage daily study tasks, track streaks, and stay motivated.

##  Features

* Add and complete study tasks
* Subject categories
* Daily streak tracker 
* Progress bar
* Confetti celebration on completion 
* Dark / Light mode
* Data saved in browser (localStorage)

##  What I Learned

* DOM manipulation
* Local storage persistence
* Date & time logic
* UI feedback systems
* State management in JavaScript

##  How to Run

Download the project and open `index.html` in any browser.
